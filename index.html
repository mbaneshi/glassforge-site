What GlassForge detects

UI Stack Analysis

UIKit vs SwiftUI file counts, storyboard/XIB inventory, and Liquid Glass design signals.

Deprecated API Detection

Pattern-based scanning against versioned iOS 26 rules. Every finding includes a replacement suggestion.

Dependency Risk Scoring

CocoaPods to SPM migration mapping with difficulty ratings and risk scores per dependency.

Objective-C Footprint

Mixed-module detection, bridging header analysis, and ObjC-to-Swift migration surface estimation.

Layout Risk Signals

Hard-coded frames, fixed constants, and patterns that won't adapt to Liquid Glass dynamic layouts.

CI-Friendly Output

JSON artifacts for automation. Use --fail-under as a gate in your CI pipeline.
